JavaScript String Length

Properti length menghasilkan nilai dari panjang sebuah string

Example:

Syntax:
(string).length

Result:

JavaScript String slice()

slice() method mengambil beberapa string dan mengembalikan ke string baru. Method ini memiliki 2 parameter yaitu: start position dan end position

Example:

Syntax:
(string).slice(12, 15)

Result:

JavaScript String substr()

substr() dan slice() hampir mirip. Cuma beda di parameter kedua. parameter kedua berupa panjang string yang ingin di ambil

Example:

Syntax:
(string).substr(5, 5)

Result:

Jika parameter kedua tidak diberi nilai alias null maka substr() akan mengambil seluruh panjang string

Example:

Syntax:
(string).substr(5)

Result:

JavaScript String replace()

replace() berfungsi untuk mengganti string yang di definisikan

Example:

Syntax:
(string).replace("Benci", "Cinta")

Result:

replace() method hanya mengganti string pada saat string pertama ditemukan

Example:

Syntax:
(string).replace("Benci", "Cinta")

Result:

replace() method juga bersifat Case Sensitive artinya, setiap huruf besar dan kecil berpengaruh pada pencarian

Example:

Syntax:
(string).replace("Benci", "Cinta")

Result:
