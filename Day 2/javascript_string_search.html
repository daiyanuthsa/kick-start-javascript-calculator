JavaScript String indexOf()

indexOf() method berfungsi mengembalikan nilai index dari posisi string pertama yang di ditemukan di dalam text

Example:

Syntax:
(string).indexOf("budi")

Result:

Perlu di ingat!, indexOf() hanya mengembalikan nilai index pada string pertama yang ditemukan

JavaScript lastIndexOf()

JavaScript includes()

includes() method berfungsi untuk mengembalikan nilai true jika string ditemukan di dalam text, sebaliknya bernilai false jika tidak ditemukan

Example:

Syntax:

JavaScript startWith()
