Back-Tics Syntax

Template literals mendefinisikan sebuah string menggunakan Back-Tics (``) syntax

Example:
var txt = `Ini adalah string`

Quote inside String

Bisa juga menggunakan gabungan dari Double Quote dan Single Quote

Example:
var txt = `Ini adalah sebuah "string"`

Multiline Strings

Template literals juga bisa membuat multi line string

Example:
var txt =
`ini
adalah
string`

Interpolation

Template literals memudahkan juga dalam meng-interpolasi variable dan expression

Syntax:
${...}

Variable Subtitution

Template literals bisa memanggil variable dengan Interpolation

Example:
var firstName = "Muhammad"
var lastName = "Irfan"
var result = `Hi ${firstName} ${lastName}`

Expression Subtitution

Template literals bisa memanggil expression dengan Interpolation

Example:
var harga = 100
var diskon = 0.3
var result = `Total Diskon ${(harga * diskon).toFixed(2)}`
