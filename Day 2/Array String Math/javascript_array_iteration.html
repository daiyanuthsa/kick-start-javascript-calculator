JavaScript Array forEach()

forEach() mengoperasikan serta memanggil callback function pada setiap elemen

Example:
var txt = ""
var arr = [24, 38, 255, 48]

Callback function:
function addToTxt(value, index, array) { txt += value}

JavaScript Array map()

Sedikit berbeda dari forEach(). map() membuat array baru dengan melakukan iterasi dari setiap elemen array. map() juga tidak mengganti sifat dari array sebelum nya

Example:
var arr = [1, 2, 3, 4, 5]

Callback function:
function multipleByTwo(value, index, array){ return value * 2 }

Syntax:
var newArr = (array).map(multipleByTwo)
