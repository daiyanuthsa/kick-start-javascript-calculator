Converting Array to String

Array bisa di konversi kedalam bentuk string dengan menggunakan method toString()

Example:
var arr = [1, "B", "C"]

Syntax:
(array).toString()

JavaScript Array pop() and push()

Array pop()

Array pop() digunakan untuk menghapus element terakhir dari sebuah array

Example:
var arr = ["A", "B", 123]

Syntax:
arr.pop()

Result:

Array push()

Array push() digunakan untuk menambah element baru terakhir dari sebuah array

Example:
var arr = ["A", "B", 123]

Syntax:
arr.push("C")

JavaScript Array concat() (Mergering)

concat() digunakan untuk menggabungkan 2 array ke dalam array baru

Example:
var pria = ["Shandy", "Rizal", "Mirza"]
var wanita = ["Putri", "Ziana", "Angel"]

Syntax:
var kumpulan = pria.concat(wanita)
