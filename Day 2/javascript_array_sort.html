JavaScript Array sort()

sort() digunakan untuk mengurutkan array secara alphabet mulai dari terkecil hingga terbesar

Example:
var arr = ["Semangka", "Mangga", "Apel"]

Syntax:
var result = arr.sort()

JavaScript Array sort() Numeric

Example

Syntax:
