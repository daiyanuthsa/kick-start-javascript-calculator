Creating Array

Ada 2 cara untuk mendefinisikan Array

Example 1:
var arr = ["A", "B", "C"]

Example 2:
var arr = []
arr[0] = "A"
arr[1] = "B"
arr[2] = "C"

Creating Array using Keyword New

Kemudian ada cara lain mendefinisikan array menggunakan keyword new

Example:
var arr = new Array("A", "B", "C")

Accessing Array Elements

Untuk mengakses Array Element, harus di definisikan index dari element yg di minta

Example:
var arr = ["A", "B", "C"]

Syntax:
var a = arr[0]
